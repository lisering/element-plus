# Title 标题

标题样式，常用于页面布局区域标题。

## 基础用法

设置title属性来设置标题。

:::demo 通过设置 `title` 属性来设置标题内容。

title/basic

:::

### Title 属性

| 名称         | 说明             | 类型                          | 默认值     | 必填 |
| ---------- | -------------- | --------------------------- | ------- | -- |
| `title`   | 标题名称           | ^[string]                   | `""`     | 是  |
